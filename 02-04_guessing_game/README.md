# Comparing the guess to the secret number

The guessing game now parses the input into a `u32` and compares it against the secret number
using `Ordering` and a `match` expression, printing whether the guess was too small, too big, or
correct.

Docs: https://doc.rust-lang.org/book/ch02-00-guessing-game-tutorial.html#comparing-the-guess-to-the-secret-number
